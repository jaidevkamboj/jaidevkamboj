Hi 👋

I'm **Jaidev Kamboj**, a passionate **JUCE Developer, Audio Software Engineer, DSP Engineer and MATLAB Developer** from India.

I specialize in designing and developing professional audio applications, real-time DSP systems, cross-platform audio plugins, and live sound processing solutions using JUCE Framework, C++, and MATLAB.
